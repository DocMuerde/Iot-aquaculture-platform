# Main Board v1

## Description
Main PCB of the IoT aquaculture device.

## Features
- ESP32-based controller
- Analog sensor interface (pH, EC, O2)
- Digital sensors (DHT22, DS18B20)
- MicroSD interface
- Relay outputs
